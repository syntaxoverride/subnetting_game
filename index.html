Subnetting Help

Subnet Calculation Steps

  1. Identify the network address and CIDR prefix (e.g., 192.168.1.0/24)
  2. Determine required hosts - How many hosts need to be accommodated
  3. Calculate the appropriate subnet mask - Based on the number of hosts
  4. Find the network address - This is the base address of the subnet
  5. Find the first usable IP - Network address + 1
  6. Find the broadcast address - Last address in the subnet
  7. Find the last usable IP - Broadcast address - 1

Subnet Mask Tips

To find the subnet that can accommodate N hosts:

  • Calculate bits needed: log₂(N+2) rounded up
  • CIDR prefix = 32 - bits needed
  • Example: For 12 hosts, you need 4 bits (2⁴=16), so /28 CIDR
